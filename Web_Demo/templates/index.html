About This Project

Music Genre Classification

This web application uses machine learning algorithms to analyze and classify music files into different genres. The classification is based on various audio features extracted from the music file.

How It Works

  1. Upload your music file (MP3, WAV, OGG, etc.)
  2. Our system analyzes the audio features
  3. Machine learning algorithms classify the music
  4. The predicted genre is displayed

Supported Genres

Rock Pop Hip Hop Jazz Classical Electronic R&B Country Reggae Metal

Technology

This application is built using HTML, CSS, and JavaScript for the frontend. The genre classification is performed using a machine learning model trained on thousands of music samples across different genres.
